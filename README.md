💣 BombAll
Welcome to the BombAll repository! This is an action game developed using the LÖVE (Love2D) framework.

🎮 How to Run
Option 1: Quick Start (Windows)
Download the latest version from the Releases page.

Extract the archive to any folder.

Run BombAll.exe.

Option 2: Run from Source (For Developers)
If you have LÖVE installed on your system:

Clone this repository: git clone https://github.com/YOUR_USERNAME/BombAll.git

Navigate to the project folder.

Run the game using the command: love .
